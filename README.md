# 🐱 HaChiCat — 桌面宠物助手

一只会陪你工作、帮你干活的桌面小猫。

它不是聊天机器人——它是一只有个性的桌面宠物，平时安静地趴在屏幕上，你需要的时候召唤它，帮你记录笔记、管理待办、翻译文字、搜索网页。

---

## ✨ 它能做什么

### 🐾 活着的桌面宠物

- **一直显示在桌面上**，半透明无边框，不遮挡你的工作
- **点击拖动**它可以随便移动，**快速甩出去**会有物理惯性（速度、摩擦、边缘弹跳）
- **点一下会"哈~"** —— 白色小气泡从头顶冒出，慢悠悠飘走
- **对你的操作有反应** —— 你在别处点鼠标、敲键盘，它会闪烁一下，好像在关注你
- **自动散步** —— 它会自己在屏幕上游荡，像真的宠物一样
- 右键宠物弹出菜单：查看待办、查看笔记、图库、⏮⏯⏭ 音乐三键、隐藏
- 大小可调（10% ~ 250%），可以换皮肤

### 🧠 选中文字 → 一键处理

在任何地方选中文字，按 `Ctrl+Shift+A`（可在设置中自定义快捷键），会弹出一个浮动菜单：

| 功能 | 说明 |
|------|------|
| 🔍 **搜索** | 用浏览器搜索选中的文字，支持 Bing / Google / 百度 / DuckDuckGo |
| 📝 **记笔记** | 保存选中文字为笔记，支持 Markdown 渲染 |
| 📋 **添加待办** | AI 自动从文字中提取待办事项，支持多条待办、智能时间识别 |
| 🌐 **翻译** | 英译中，单词以词典格式展示（词性 + 多释义），句子直接翻译 |

AI 处理时宠物会冒出 "..." 思考气泡，不会卡住。

### 📝 笔记管理

- 卡片式布局，展开后可以编辑全文
- **Markdown 渲染开关** —— 一键切换源码 / 排版预览
- 自动保存，不用担心丢失
- 支持自定义背景图片

### ✅ 待办管理

- AI 帮你把一段话自动提炼成待办标题
- 支持**多条待办同时提取**
- **时间智能识别** —— "明天下午三点"、"下周五" 都能正确解析
- 编辑面板可修改标题、截止日期和原文
- 完成后打勾归档

### 🖼 图库

- 复制图片后按快捷键可以保存到图库
- 网格浏览，查看大图（滚轮缩放，中心锚点）、复制、删除
- 缩略图缓存，滚动流畅

### 🎵 音乐控制

- 右键菜单直接控制：⏮ 上一首 / ⏯ 播放暂停 / ⏭ 下一首
- 模拟键盘媒体键，兼容大多数播放器

### ⏰ 主动提醒

- 宠物主动提醒待办事项和喝水休息
- 紧急程度自适应，间隔随机 ±5~30 分钟，不会机械重复

### ⚙️ 设置

- **LLM 配置**：支持 DeepSeek / OpenAI / Ollama，一键检测可用模型
- **快捷键自定义**：点击捕获组合键，保存即生效
- 所有设置实时生效，无需重启

## 🚀 安装运行

### 环境要求
- **Python 3.11+**
- **Windows 10/11**

### 步骤

```bash
git clone <仓库地址>
cd HaChiCat
python -m venv venv
venv\Scripts\activate
pip install PySide6 pydantic openai httpx
python run.py
```

### 配置 AI

1. 去 [platform.deepseek.com](https://platform.deepseek.com) 注册获取 API Key
2. 右键托盘图标 → 设置，填入 API Key，点「检测模型」自动拉取可用模型
3. 也可选择 Ollama 本地模型或 OpenAI

不用 AI 的话，搜索、笔记、图库、音乐等功能不受影响。

### 打包

```bash
pip install pyinstaller
pyinstaller --clean HaChiCat.spec
# 输出在 dist/HaChiCat/HaChiCat.exe
```

## 🎮 操作一览

| 操作 | 方法 |
|------|------|
| 移动宠物 | 按住左键拖动 |
| 扔出宠物 | 拖动后快速松手 |
| 气泡互动 | 单击左键——"哈~"气泡冒出 |
| 右键菜单 | 右键点击宠物 |
| 召唤功能菜单 | 选中文字 → `Ctrl+Shift+A` |
| 设置 | 托盘图标右键 → 设置 |
| 退出 | 托盘图标右键 → 退出 |

## 🏗 项目结构

```
HaChiCat/
├── run.py                 # 入口
├── src/
│   ├── main.py            # 主程序、信号调度、Agent 管线
│   ├── pet/               # 宠物窗口、渲染、动画、物理、气泡、状态机
│   ├── agent/             # LLM 客户端、意图分类、提醒引擎
│   ├── tools/             # 笔记、待办、图库、音乐、翻译、图片查看器
│   ├── input/             # 全局热键管理、被动输入监听
│   ├── memory/            # SQLite 数据库、Pydantic 配置
│   └── utils/             # 主题设计 token、Markdown→HTML 转换、日志
├── assets/                # 精灵图、图标
├── img/                   # 宠物图片（支持多张切换）
└── data/                  # 运行时数据（自动生成，gitignore）
```
