# 🐱 HaChiCat — 桌面宠物助手

一只会陪你工作、帮你干活的桌面小猫。

它不是聊天机器人——它是一只有个性的桌面宠物，平时安静地趴在屏幕上，你需要的时候召唤它，帮你记录笔记、管理待办、翻译文字、搜索网页。

---

## ✨ 功能特性

### 🐾 桌面宠物

- 半透明无边框，一直显示在桌面上
- **点击拖动**移动，**快速甩出**有物理惯性（速度、摩擦、边缘弹跳）
- **点一下冒"哈~"** —— 纯白气泡从头顶慢悠悠飘出
- **对你的操作有反应** —— 在别处点鼠标或敲键盘，它会闪烁一下
- **哈气模式** —— 鼠标靠近时自动推开，不让点
- **任务栏吸附** —— 拖到任务栏附近自动贴紧
- **番茄钟** —— 倒计时气泡，完成后大量"哈~"庆祝
- 右键菜单：待办、笔记、图库、音乐控制、番茄钟、隐藏
- 大小可调（10% ~ 250%）

### 🎨 多形象支持

- **HaChiCat 默认形象** —— 自带猫猫，点击闪烁切换表情
- **Petdex 形象库** —— 支持 3400+ 社区动画宠物，一键下载安装
- 8 帧动画状态（idle/wave/run/failed/review/jump 等）
- AI 任务完成时挥手庆祝，失败时沮丧，处理时思考
- 水平翻转、删除等管理功能

### 🧠 AI 助手 (Ctrl+Shift+A)

选中任何文字，按快捷键弹出浮动菜单：

| 功能 | 说明 |
|------|------|
| 🔍 **搜索** | 浏览器搜索，支持 Bing/Google/百度/DuckDuckGo |
| 📝 **记笔记** | 保存文字，支持 Markdown 渲染预览 |
| 📋 **添加待办** | AI 自动提取待办，支持多条识别、智能时间解析 |
| 🌐 **翻译** | 英译中，单词词典格式（词性+多释义），句子流畅翻译 |

### 📝 笔记 & 待办

- 卡片式布局，Markdown 渲染开关
- AI 自动提炼待办标题，支持多条同时提取
- 智能时间识别（"明天下午三点"、"下周五"）
- 编辑面板可修改标题、截止日期、原文
- 自动保存，自定义背景图

### 🖼 图库 & 音乐

- 复制图片 → 快捷键保存到图库
- 网格浏览，滚轮缩放（中心锚点），复制/删除
- 右键菜单一键控制音乐播放（模拟媒体键）

### ⚙️ 设置

- LLM 配置：支持 DeepSeek / OpenAI / Ollama，一键检测可用模型
- 快捷键自定义：点击捕获组合键
- 所有设置实时生效，无需重启

## 🚀 快速开始

### 环境要求

- **Python 3.11+**
- **Windows 10/11**（使用了 Win32 API）

### 安装运行

```bash
# 克隆项目
git clone https://github.com/sduwqh/HachiCat.git
cd HaChiCat

# 创建虚拟环境
python -m venv venv
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 运行
python run.py
```

首次运行会自动创建 `data/` 目录和 SQLite 数据库。

### 配置 AI 功能

翻译、待办提取、提醒等需要大模型 API：

1. 注册 [DeepSeek](https://platform.deepseek.com) 获取 API Key
2. 右键托盘图标 → 设置 → 填入 API Key
3. 点击「检测模型」自动拉取可用模型列表
4. 也可在设置中切换到 OpenAI 或 Ollama

不用 AI 的话，搜索、笔记、图库、音乐等功能完全不受影响。

### 安装 Petdex 形象

1. 设置 → 切换形象 → **+ 添加新形象**
2. 在 [petdex.dev](https://petdex.dev) 挑选喜欢的宠物
3. 点击 Copy Page Link 复制链接
4. 粘贴到输入框 → 自动下载安装 → 永久可用

### 打包成 exe

```bash
pip install pyinstaller
pyinstaller --clean HaChiCat.spec
# 输出在 dist/HaChiCat/HaChiCat.exe
```

## 🎮 操作一览

| 操作 | 方法 |
|------|------|
| 移动宠物 | 按住左键拖动 |
| 扔出宠物 | 拖动后快速松手 |
| "哈~"气泡 | 点击左键 |
| 右键菜单 | 右键点击宠物 |
| 召唤功能菜单 | 选中文字 → `Ctrl+Shift+A` |
| 设置 | 托盘图标右键 → 设置 |
| 退出 | 托盘图标右键 → 退出 |

## 🏗 项目结构

```
HaChiCat/
├── run.py                 # 入口
├── requirements.txt       # 依赖
├── src/
│   ├── main.py            # 主程序、Agent 管线、信号调度
│   ├── pet/               # 宠物窗口、渲染、动画、物理、气泡
│   ├── agent/             # LLM 客户端、意图分类、提醒引擎
│   ├── tools/             # 笔记、待办、图库、音乐、翻译
│   ├── input/             # 全局热键、被动输入监听
│   ├── memory/            # SQLite 数据库、配置管理
│   └── utils/             # 主题 token、Markdown→HTML、宠物扫描
├── img/                   # 宠物图片资源
├── assets/                # 精灵图、图标
├── pets/                  # 下载的 Petdex 形象（gitignore）
└── data/                  # 运行时数据（自动生成，gitignore）
```
